# FIBONACCO 90-DAY CONTENT PLAN
## Complete Outline - 60 Campaigns

---

# THE STRUCTURE

## Campaign Types:
- **EDU (15)**: Educational presentations - teach a concept
- **HOOK (15)**: Action triggers - "we built this for you"
- **HOWTO (30)**: Implementation guides - step-by-step

## The Funnel Logic:
```
WEEK PATTERN (Typical):
Day 1: HOOK - Immediate action opportunity
Day 2: EDU - Teach the concept behind the hook
Day 3: HOWTO - How to implement
Day 4: HOWTO - Related implementation
Day 5: HOWTO - Advanced implementation
```

## Content Pillars:
1. **THE NEW DAY** - Renaissance of community marketing
2. **AI DISCOVERY** - How people find businesses now
3. **AI FOR BUSINESS** - Leverage AI to compete
4. **THE PLATFORM** - How Fibonacco works
5. **THE PLAYBOOKS** - Strategic guides by outcome
