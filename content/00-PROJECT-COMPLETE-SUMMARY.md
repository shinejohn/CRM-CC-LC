## WHAT EACH CAMPAIGN INCLUDES

Every campaign JSON file contains:

1. **Campaign Metadata**
   - ID, week, day, type
   - Title, subject line
   - Landing page reference

2. **Complete Slide Deck (6-9 slides)**
   - Component type
   - Title and content
   - Full narration script
   - Duration timing
   - Audio file reference

3. **Email Sequence**
   - Subject lines
   - Preview text
   - Complete body copy
   - Personalization tokens

4. **Campaign Connections**
   - leads_to: next recommended campaign
   - follows: previous campaign in sequence
